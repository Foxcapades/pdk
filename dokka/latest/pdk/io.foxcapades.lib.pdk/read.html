fun InputStream.read(deque: ByteDeque): Int
Content copied to clipboard

Reads bytes into the tail end of the given ByteDeque until either the input stream has no more data or the given deque has no more space available.

If the given deque has no room available (indicated by a ByteDeque.space value of 0) this method does nothing and returns 0 to indicate that no bytes were read.

This function is based on InputStream.read and thus will block until data is available, until the end of the stream has been reached, or until an exception is thrown.

If the receiver stream has no data remaining when this function is called, this function will return -1 to indicate that there were no bytes in the stream to be read.

Otherwise, if the stream does have input and the given deque does have room, this method returns the number of bytes read into the given deque.

Return

The total number of bytes read into the given deque, or -1 to indicate the end of the stream had already been reached.

Parameters

deque

Deque into which data from the receiver InputStream will be read.
